# 📋 Requisitos do Sistema

## Software Necessário

### Backend
- **Python**: 3.8 ou superior
- **pip**: Gestor de pacotes Python
- **DuckDB**: Instalado automaticamente via requirements.txt

### Frontend
- **Node.js**: 18.0 ou superior
- **npm**: 9.0 ou superior (vem com Node.js)

### Opcional
- **Git**: Para controlo de versão
- **Docker**: Para containerização (futuro)

## Verificar Instalações

### Python
```bash
python --version
# Deve mostrar: Python 3.8.x ou superior
```

### Node.js
```bash
node --version
# Deve mostrar: v18.x.x ou superior
```

### npm
```bash
npm --version
# Deve mostrar: 9.x.x ou superior
```
